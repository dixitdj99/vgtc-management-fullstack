# Advanced Weather Header

A modern, responsive website header with real-time weather data and highly realistic animations.

## Features
- **Live Data**: Fetches real-time weather from WeatherAPI.com.
- **Auto-Location**: Detects user location via browser geolocation with IP fallback.
- **Realistic Animations**: Custom CSS animations for Clear, Cloudy, Rain, Thunderstorm, Snow, Mist, and more.
- **Glassmorphism UI**: Modern aesthetic with backdrop blur and responsive layout.
- **Unit Toggle**: Click the temperature to toggle between Celsius and Fahrenheit.
- **Performance Optimized**: Uses lightweight CSS keyframes and radial gradients.

## Quick Start
1. Clone or download this folder.
2. Open `index.html` in your browser.
3. *Optional*: If geolocation is blocked, the app will fallback to auto-IP detection.

## Technical Details
- **HTML5**: Semantic structure.
- **CSS3**: Advanced gradients and keyframe animations.
- **Vanilla JS**: No heavy libraries required.
- **WeatherAPI**: Integrated via Fetch API with the provided key.

## Weather States
Use the manual controls in the demo area to preview different weather states:
- **Clear**: Sun pulse effect.
- **Clouds**: Multi-layered parallax cloud drift.
- **Rain**: High-speed vertical drops.
- **Storm**: Randomized lightning strikes + rain.
- **Snow**: Blurry particle fall.
- **Night**: Twinkling stars + moon glow.
- **Sunset**: Vibrant orange-purple gradient.
